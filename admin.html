Generate Access Codes



Instructions:

  1. Click "GENERATE CODES" to create new access codes
  2. Copy the hash (not the code itself) from each generated code
  3. Add each hash to the validCodes Set in access.html line 94
  4. Share only the CODE with users (never share the hash)
  5. Each code can only be used once
  6. Codes are case-insensitive

Example:

If you generate code A7X9K2 with hash abc123...

Add to access.html:

"abc123...",

Give user: A7X9K2
